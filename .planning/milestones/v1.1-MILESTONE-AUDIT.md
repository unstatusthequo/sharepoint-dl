## Bugs Fixed During UAT & Audit
- Playwright driver mismatch on Node.js upgrade (run.sh health check)
- Cancel flow showing bogus "Done!" with 0 files
- Config not saving when download has partial failures
- `&` in filenames causing download failures (URL encoding)
- Files commingled with metadata (timestamped subdirectory)
- Long filenames pushing progress bar off-screen
- Verify command failing on legacy double-prefixed paths
- SharePoint .xlsx recompression causing false size-mismatch failures
- Process hanging after verification (thread cleanup)
- Authenticated sharing links (/:f:/r/) not resolving site URL correctly
- Parent-domain cookies (.sharepoint.com) not loaded for session auth
- Completed file counts inflated by stale state.json from prior runs
- Flat mode overwriting duplicate filenames (now prompts for layout)
- Throttle not persisted in CLI download command's save_config
- Worker display slot allocation race condition (thread-based slots)
- Overall progress not finalized to 100% when workers finish
