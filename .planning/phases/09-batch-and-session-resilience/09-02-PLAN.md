    1. Add `on_auth_expired: Callable[[], bool] | None = None` parameter to `download_all()` signature (after `throttle`). This is backward-compatible since it defaults to None.

    2. Modify the `except AuthExpiredError` block inside the `worker()` closure (around line 291-296). Change from:
       ```python
       except AuthExpiredError as e:
           auth_halt.set()
           auth_error.append(e)
           state.set_status(url, FileStatus.FAILED, error="auth_expired")
           logger.error("Failed: %s -- auth expired", file_entry.name)

       ```
       To:
       ```python
       except AuthExpiredError as e:
           state.set_status(url, FileStatus.FAILED, error="auth_expired")
           logger.error("Failed: %s -- auth expired", file_entry.name)
           if on_auth_expired is not None:
               auth_halt.set()  # Pause new submissions
               refreshed = on_auth_expired()
               if refreshed:
                   auth_halt.clear()  # Resume workers
                   return  # File stays FAILED — retry loop picks it up
           # No callback or refresh failed — original abort behavior
           auth_halt.set()
           auth_error.append(e)
           raise
       ```

       - `test_on_auth_expired_false_aborts`: Verify AuthExpiredError propagated when callback returns False

    5. Update the `Callable` import in engine.py's TYPE_CHECKING block if not already present (it is — line 14 imports Callable).
