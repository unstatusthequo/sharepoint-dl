    **Logging module (REL-03):**
    Create `sharepoint_dl/downloader/log.py` with two public functions:

    `setup_download_logger(dest_dir: Path) -> logging.Logger`:
    - Get or create logger named `"sharepoint_dl"` (the package root logger)
    - Remove any existing FileHandlers on it (idempotent setup)
    - Create `FileHandler(dest_dir / "download.log", mode="a")`
    - Set formatter: `"%(asctime)s | %(levelname)s | %(message)s"` with datefmt `"%Y-%m-%d %H:%M:%S"`
    - Set level to `logging.DEBUG` on the handler (logger level INFO)
    - Set `logger.propagate = False` (prevents any root logger StreamHandlers from firing)
    - Return the logger

    `shutdown_download_logger()`:
    - Get logger `"sharepoint_dl"`
    - Remove and close all FileHandlers
    - This ensures file handles are released after download completes

    Write tests in `tests/test_logging.py` using tmp_path fixture. Write ETA column test in `tests/test_downloader.py`.
