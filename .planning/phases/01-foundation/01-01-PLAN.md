  <behavior>
    - harvest_session(url, timeout=120) opens headed Chromium, navigates to url, polls cookies every 2s for FedAuth/rtFa, calls storageState(), closes browser, returns session path
    - harvest_session raises TimeoutError after timeout_seconds with clear message
    - save_session(storage_state_path, sharepoint_url) adds _host to state, writes to ~/.sharepoint-dl/session.json with 0o600 permissions
    - load_session(sharepoint_url) returns requests.Session with cookies if session.json exists and host matches; None otherwise
    - build_session(session_path, sharepoint_host) injects all matching-domain cookies into requests.Session
    - validate_session(session, site_url) GETs _api/web/title, returns True on 200, False on 401/403/exception
  </behavior>
  <action>
    1. Implement `sharepoint_dl/auth/browser.py`:
       - `harvest_session(sharepoint_url: str, timeout_seconds: int = 120) -> Path`
       - Uses `sync_playwright`, launches Chromium with `headless=False` (mandatory — user must see browser for MFA/OTP)
       - Navigates to sharepoint_url
       - Polls `context.cookies()` every 2 seconds checking for "FedAuth" or "rtFa" cookie names
       - When found: calls `context.storage_state(path=str(tmp_path))`, then calls `save_session(tmp_path, sharepoint_url)`
       - On timeout: closes browser, raises `TimeoutError(f"Authentication not detected within {timeout_seconds}s. Open {sharepoint_url} manually to check the login flow.")`
       - Always closes browser in finally block

    2. Implement `sharepoint_dl/auth/session.py`:
       - `SESSION_DIR = Path.home() / ".sharepoint-dl"`
       - `save_session(storage_state_path: Path, sharepoint_url: str) -> Path` — reads storageState JSON, adds `"_host": urlparse(sharepoint_url).netloc`, writes to SESSION_DIR/session.json, sets chmod 0o600, returns path
       - `load_session(sharepoint_url: str) -> requests.Session | None` — returns None if file missing or `_host` != current host; otherwise calls build_session
       - `build_session(session_path: Path, sharepoint_host: str) -> requests.Session` — reads JSON, filters cookies by domain containing sharepoint_host, injects each via `session.cookies.set(name, value, domain=domain, path=path)`
       - `validate_session(session: requests.Session, site_url: str) -> bool` — GET `{site_url}/_api/web/title` with `Accept: application/json;odata=verbose` header, timeout=(10, 30), returns `resp.status_code == 200`; returns False on any RequestException

    3. Update `sharepoint_dl/auth/__init__.py` to export: `harvest_session`, `load_session`, `save_session`, `build_session`, `validate_session`

       - Ensure test_harvest_session_writes_file mocks the full Playwright chain: sync_playwright -> chromium.launch -> new_context -> new_page -> page.goto -> context.cookies -> context.storage_state
