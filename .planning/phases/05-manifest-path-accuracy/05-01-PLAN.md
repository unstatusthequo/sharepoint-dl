<objective>
Make manifest path evidence match the actual on-disk output path for every completed file.

Purpose: The remaining v1 blocker is not whether a manifest exists; it is whether the manifest's `local_path` is trustworthy. Phase 4 already persisted the real relative output path in `state.json`, so this phase makes that persisted value the manifest source of truth and adds one shared fallback path derivation for legacy state.
Output: accurate `manifest.json` local paths in normal, flat, legacy, and auth-expired partial-run cases, with regression coverage.
</objective>
