<task type="auto" tdd="true">
  <name>Task 1: Verify manifest module</name>
  <files>sharepoint_dl/manifest/verifier.py, tests/test_verifier.py</files>
  <behavior>
    - verify_manifest(dest_dir) with all files matching returns (results, 0 failures) where each result has status="PASS"
    - verify_manifest(dest_dir) with a tampered file returns status="FAIL" with expected vs actual hash
    - verify_manifest(dest_dir) with a missing file returns status="MISSING"
    - verify_manifest(dest_dir) with no manifest.json raises FileNotFoundError
    - Returns list of VerifyResult(name, local_path, expected_sha256, actual_sha256, status) plus summary counts
  </behavior>
  <action>
    Create `sharepoint_dl/manifest/verifier.py` per CONTEXT.md decisions:

    1. `class VerifyResult(NamedTuple)`:
       - `name: str`
       - `local_path: str`
       - `expected_sha256: str`
       - `actual_sha256: str | None` (None if file missing)
       - `status: str` ("PASS", "FAIL", "MISSING")

    2. `class VerifySummary(NamedTuple)`:
       - `results: list[VerifyResult]`
       - `total: int`
       - `passed: int`
       - `failed: int`
       - `missing: int`

    3. `def verify_manifest(dest_dir: Path, on_progress: Callable[[str, int], None] | None = None) -> VerifySummary`:
       - Read `dest_dir / "manifest.json"`, parse JSON. Raise FileNotFoundError if not found.
       - For each file entry in manifest["files"]:
         - Resolve `local_path` relative to `dest_dir`.
         - If file doesn't exist on disk: status=MISSING, actual_sha256=None.
         - If file exists: read in 8MB chunks, compute SHA-256, compare to `entry["sha256"]`.
         - If match: PASS. If mismatch: FAIL.
         - Call `on_progress(name, size_bytes)` after each file for progress bar updates.
       - Return VerifySummary with counts.

    4. Extra files on disk but not in manifest: IGNORED per CONTEXT.md decision.

    Tests in `tests/test_verifier.py` using tmp_path:
    - Create a fake manifest.json with 2-3 files, write matching files -> all PASS.
    - Tamper one file -> that file FAIL, others PASS.
    - Delete one file -> MISSING.
    - No manifest.json -> FileNotFoundError.
  </action>

  </verify>
  <done>Verifier reads manifest, re-hashes files from disk, reports PASS/FAIL/MISSING per file with SHA-256 comparison. All test scenarios pass.</done>
